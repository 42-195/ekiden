#  EKIDEN 
EKIDEN取自日本箱根驿传（马拉松接力赛），投资是一件不容易的事，更多时候需要坚守、坚持，就好像一场马拉松。

这是价投的时代，希望你我都可以在这个时代赚到属于自己钱。

本项目将采集A股上市公司历史财报数据，通过算法分析公司经营情况，判断企业价值、管理效率、盈利能力、抗风险能力和未来发展前景，为投资者排除垃圾公司，选出好公司。

## 一、你能从本项目中获取什么
1. 作为前端开发工程师的一次技术尝试，学习react、koa、数据开发等技术
2. 作为投资者的一次学习财务报表分析机会，学会识别好公司，避免踩雷
3. 希望在世界各个角落的我们通过本项目能够相识，未来一起经历

## 二、技术选型
react、koa、mongo、docker
## 三、业务基础架构
1. 定时任务：从互联网采集数据，进行聚合、清洗、转换，存入数据库
2. API服务：从数据库获取数据，为前端提供API接口
3. 前端服务：报表图形化解读
